# invoice-converter

쇼핑몰·결제 수단별 주문 엑셀을 택배사 업로드 양식으로 변환하는 웹 도구.

## 지원 양식

### 주문 파일 (입력)
- **카페24** — `카페 24 기본양식.csv` 형식 (수령인, 수령인 휴대전화, 수령인 주소+상세주소 등)
- **네이버페이** — 발주/발송관리에서 '엑셀 다운'으로 받은 주문 파일 (암호 설정 시 비밀번호 입력)

### 택배사 양식 (출력)
- **롯데택배** — `롯데택배송장양식.xlsx` (받는사람, 전화번호1, 우편번호, 주소, 수량(A타입), 상품명1, 배송메시지)

## 기능

- 카페24, 네이버페이 주문 파일 동시 업로드
- 롯데택배 업로드 양식으로 변환
- 네이버페이 암호화 엑셀 비밀번호 지원
- 브라우저에서만 처리 (서버 업로드 없음)

## 실행

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

## 기술 스택

- Next.js 16, React 19, TypeScript
- Tailwind CSS 4
- SheetJS (xlsx)
