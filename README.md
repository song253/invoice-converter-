# invoice-converter

쇼핑몰·결제 수단별 주문 엑셀을 택배사 업로드 양식으로 변환하는 웹 도구.

## 동작 방식

업로드한 엑셀의 **컬럼명을 자동으로 읽어** 필요한 정보만 추출한 뒤, 택배사 양식에 맞게 변환합니다. 카페24, 네이버페이 등 **출처를 구분할 필요 없이** 어떤 양식이든 사용할 수 있습니다.

## 기능

- 어떤 주문 엑셀/CSV든 업로드 (여러 파일 동시 가능)
- 컬럼명 기반 자동 필드 추출 (수령인, 연락처, 주소, 상품명 등)
- 롯데택배 업로드 양식으로 변환
- 암호화된 엑셀 비밀번호 지원
- 브라우저에서만 처리 (서버 업로드 없음)

## 실행

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

## 기술 스택

- Next.js 16, React 19, TypeScript
- Tailwind CSS 4
- SheetJS (xlsx), office-crypto
